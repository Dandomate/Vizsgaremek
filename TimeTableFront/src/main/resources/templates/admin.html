Administrator home page

This page is for users with administrative privileges. The page also has a navigation menu that is responsible for navigating between administration pages. In addition to the buttons, there is also a logout tab.

The home page summarizes the pages and their usage in boxes. Each page has a separate description or box.

Add Teacher

On the Add Teacher page, you can add teachers to the database based on the following information: -OMA ID -Name -Phone number -Password - Admin The page displays all teachers and all related data in a table. Teacher data can be changed at the touch of a button.

In addition to the Edit Data button, there is another button that is responsible for deleting the teacher. You can enter the data in forms. All forms are required. If OM is the same, our page will throw a duplication error. To add a new teacher, press the Add button. You will receive feedback on successful recording or deletion.

Add Subject

On the Add Object page, you can add objects to the database using the following information: - Name (name of the object) The page displays all objects and their associated data in a table. Subject data can be changed at the touch of a button.

In addition to the Edit Data button, there is another button that is responsible for deleting the item. You can enter the data in forms. All forms are required. Add TimeTable

On the Add timetable page, you can add a timetable to the database with the following data: Day Hour Door number Subject Teacher name Class

The page displays all the timetables in a table with all the corresponding data. The timetable data can be changed by clicking on a button. Next to the button to change the data there is another button to delete the timetable. The data can be entered in forms. All forms are mandatory. If you have the same ID, our site will throw a duplication error. To add a new timetable, you must press the Add button. You will receive a confirmation message if you have successfully added or deleted a timetable.
