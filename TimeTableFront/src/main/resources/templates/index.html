Project description

Our software is designed for institutions. Our goal was to create a personalized Timetable, in which we can store and view the data, list of classes and lessons of students, teachers and classrooms.

Institutions (schools) can store the data of students and teachers electronically. Classroom door numbers are also listed and can be viewed in the app. A list of classes and a list of students is available in our app. You can also view subjects related to teachers. The reason for creating the application was the agenda, which the user can view.

Target

Our goal was to make it easier for the Institutions to keep track of timetables and data.

It is environmentally friendly as it is available on a web interface, so there is no need to use a paper-based timetable.

It can be accessed from anywhere, so we don’t leave it like old paper timetables.

TimeTable

Scheduling of teaching time by dates and subjects.

the division of the teaching time of certain school subjects, university-college lectures, teaching sessions by days and hours.
