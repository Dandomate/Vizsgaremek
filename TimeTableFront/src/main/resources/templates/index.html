Project description

Our idea was to create a school record system, which would store the data of the school's teachers and students in a database.

It also stores the classrooms and the number of places in them, the classes and their names. It has a teacher, student and admin interface. The timetable function is also under development.

Target

Our aim was to make it easier for Institutions to keep track of timetables and data. It is environmentally friendly because it is available on a web interface, so there is no need to use paper timetables or registers.
